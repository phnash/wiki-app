# TinyMCE Keyboard Shortcuts

A complete reference guide for keyboard shortcuts available in the USB Wiki editor.

---

## Formatting Shortcuts

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `Ctrl + B` | **Bold** | Makes selected text bold |
| `Ctrl + I` | *Italic* | Makes selected text italic |
| `Ctrl + U` | <u>Underline</u> | Underlines selected text |
| `Ctrl + Shift + S` | ~~Strikethrough~~ | Strikes through selected text |
| `Ctrl + Space` | Remove formatting | Clears all formatting from selected text |

---

## Clipboard Shortcuts

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `Ctrl + C` | Copy | Copies selected text or content |
| `Ctrl + X` | Cut | Cuts selected text or content |
| `Ctrl + V` | Paste | Pastes content from clipboard |
| `Ctrl + Shift + V` | Paste as plain text | Pastes without formatting |
| `Ctrl + Z` | Undo | Reverses the last action |
| `Ctrl + Y` | Redo | Re-applies the last undone action |

---

## List Shortcuts

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `Ctrl + Shift + 7` | • Bulleted list | Creates or converts to a bulleted list |
| `Ctrl + Shift + 8` | 1. Numbered list | Creates or converts to a numbered list |
| `Ctrl + ]` | Indent | Indents a list item or paragraph |
| `Ctrl + [` | Outdent | Outdents a list item or paragraph |

## Heading Shortcuts

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `Ctrl + Alt + 1` | Heading 1 | Converts selection to Heading 1 |
| `Ctrl + Alt + 2` | Heading 2 | Converts selection to Heading 2 |
| `Ctrl + Alt + 3` | Heading 3 | Converts selection to Heading 3 |
| `Ctrl + Alt + 4` | Heading 4 | Converts selection to Heading 4 |
| `Ctrl + Alt + 5` | Heading 5 | Converts selection to Heading 5 |
| `Ctrl + Alt + 6` | Heading 6 | Converts selection to Heading 6 |
| `Ctrl + Alt + 0` | Paragraph | Converts selection back to normal text (`Ctrl + Alt + 7` also works) |

---

## Alignment Shortcuts

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `Ctrl + E` | Align centre | Centres the selected text |
| `Ctrl + L` | Align left | Aligns text to the left |
| `Ctrl + R` | Align right | Aligns text to the right |
| `Ctrl + J` | Justify | Justifies the selected text |

> **Note:** alignment applies while editing, but pages are stored as
> Markdown, which has no text-alignment syntax — alignment is not kept
> after saving.

---

## Navigation Shortcuts

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `Ctrl + Home` | Go to start | Moves cursor to the beginning of the document |
| `Ctrl + End` | Go to end | Moves cursor to the end of the document |
| `Page Up` | Scroll up | Scrolls up one page |
| `Page Down` | Scroll down | Scrolls down one page |
| `Ctrl + A` | Select all | Selects all content in the editor |

## Editing Shortcuts

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `Ctrl + F` | Find | Opens the find dialog to search within the editor |
| `Ctrl + H` | Find and replace | Opens the find and replace dialog |
| `Ctrl + Enter` | Save | Saves the page (if in edit mode) |
| `Alt + F10` | Focus toolbar | Moves keyboard focus to the editor toolbar |

## Key Terms

| Term | Definition |
| :--- | :--- |
| **Shortcut** | A combination of keyboard keys that performs a specific action |
| **Modifier Key** | A key like `Ctrl`, `Alt`, or `Shift` that is pressed in combination with another key |
| **Toggle** | A shortcut that turns a formatting option on or off (like bold or italic) |
| **Accelerator** | Another term for a keyboard shortcut |
